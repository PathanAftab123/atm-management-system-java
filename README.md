# 🏧 ATM Management System (Java + JDBC + MySQL)

A console-based ATM Management System developed using **Java, JDBC, and MySQL**.  
This project simulates real ATM operations like login, withdrawal, transfer, and account security.

---

## 🚀 Features

- 🔐 Login System with PIN
-  Account Lock after 3 Wrong Attempts
- 💰 Deposit Money
- 💸 Withdraw Money
- ⚡ Fast Cash
- 🔄 Transfer Money
- 📜 Mini Statement
- 🧾 Receipt Generation
- 📅 Daily Withdrawal Limit
- 🕒 Last Login Tracking
- 👨‍💼 Admin Panel
- 🔓 Admin Unlock Account
- 📊 Monthly Report

---

## 🛠️ Technologies Used

- Java (Core Java)
- JDBC
- MySQL Database
- IntelliJ IDEA
- Git & GitHub
